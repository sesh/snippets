Just a collection or reusable snippets that aren't quite worthy of being their own package.


## Usage

Copy & paste the snippets into your codebase.


## Development

Run everything before committing:

```
just all
```

Run tests for all snippets with tests:

```
just test
```

Format everything with `black`:

```
just format
```

Check with `bandit`:

```
just bandit
```
